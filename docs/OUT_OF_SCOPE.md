# Tính năng Out-of-Scope — Phase 1

Các tính năng sau **KHÔNG** thuộc phase 1. Nếu muốn làm phải được duyệt và lên phase sau.

## Quản lý hiện trường
- [ ] Safety management (an toàn lao động, incident report)
- [ ] QA/QC (kiểm tra chất lượng, inspection)
- [ ] Inventory management (quản lý vật tư, kho)
- [ ] Equipment management (quản lý thiết bị, máy móc)

## Quy trình phức tạp
- [ ] Multi-level approval workflow (duyệt nhiều tầng)
- [ ] Complex notification system (push notification, email template phức tạp)
- [ ] Escalation rules

## Dashboard & Báo cáo nâng cao
- [ ] Dashboard nâng cao (S-curve, Earned Value Analysis)
- [ ] Export báo cáo PDF/Excel tự động
- [ ] Custom report builder
- [ ] AI Copilot / dự đoán tiến độ

## Tích hợp bên thứ 3
- [ ] Tích hợp phần mềm kế toán
- [ ] Tích hợp BIM
- [ ] Tích hợp GPS / bản đồ real-time
- [ ] SSO / OAuth với hệ thống khác

## Khác
- [ ] Multi-language (hiện tại chỉ tiếng Việt)
- [ ] Dark mode
- [ ] Mobile native app (chỉ mobile web responsive)
- [ ] Offline mode
